# ISMRAN_Analysis
Basic analysis of ISMRAN data  

Prerequisites  
GCC : 9.4 (Ubuntu 20)  
ROOT :  6.22  
CMAKE


#Steps to Compile  
1) Download the code.  
2) Create a build directory  
   mkdir build && cd build  
3) cmake ..  
4) make -j4  

This will create many executable, whose source code can be seen the **test** directory.
